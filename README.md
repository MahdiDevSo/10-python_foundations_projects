
# 📘 10 - Python Foundations Projects

This repository contains **10 simple real-world Python projects** built using:

- Python Basics (Variables, Input/Output)
- Operators (+, -, *, /, %, ==, >, <)
- Conditions (if / elif / else)
- Logical Operators (and, or)

These projects are designed for **beginners** to practice and understand core programming concepts.

---

## 🚀 Projects List

### 1. Age Checker
Check if a person is an adult or minor.

### 2. Even or Odd Number
Determine if a number is even or odd.

### 3. Simple Login System
Basic username and password validation.

### 4. Grade Calculator
Convert marks into grades (A, B, C, Fail).

### 5. Temperature Checker
Classify weather as Hot, Normal, or Cold.

### 6. Simple Calculator
Perform basic arithmetic operations.

### 7. Voting Eligibility
Check if a user can vote based on age.

### 8. Discount System
Apply discount based on price.

### 9. Traffic Light System
Simulate traffic light behavior.

### 10. Number Comparison
Compare two numbers and find the largest.

---

## 🧠 What You Will Learn

- How to take user input (`input()`)
- How to display output (`print()`)
- How to use variables
- How to write conditions (`if`, `elif`, `else`)
- How to use operators
- How to solve real-life problems with code
